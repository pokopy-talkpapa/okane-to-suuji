# あさかつ スタンプカード — セットアップ手順

朝活（Zoom）に来てくれた子に、ぽこぴーさんがスタンプを押す → 子ども・保護者がオンラインで自分のカードを見て喜べる、をオンラインだけで完結させる仕組みです。

- **子ども向けカード**: `index.html` … あいことばを入れると自分のカードが表示される
- **管理ページ**: `admin.html` … 朝活中に名前をタップしてスタンプを押す（ぽこぴーさん専用）
- **バックエンド**: Google スプレッドシート + Google Apps Script（`gas/Code.gs`）… **すべて無料**

かかる費用: **ゼロ**（Google アカウントと GitHub アカウントがあればOK）。所要時間: だいたい10〜15分です。

> **Google スプレッドシート／Apps Script の作成とデプロイは、あなたの Google アカウントでの承認が必要な作業です**（他人が代行できません）。下の手順どおりに進めれば数分で終わります。

---

## まず動きを見たいとき（デプロイ不要）

`index.html` や `admin.html` を `?demo=1` を付けて開くと、サンプルデータで見た目と操作を試せます。
（例: `https://＜あなたのURL＞/stamp/index.html?demo=1` / `admin.html?demo=1`）
本番のデータは保存されません。仕組みを触って確認する用です。

---

## ステップ1: バックエンド（スプレッドシート）を作る

シートの中身は `setup()` が自動で作ります。手でシートを組む必要はありません。

1. [Google スプレッドシート](https://sheets.new) で**空のシートを新規作成**（名前は何でもOK、例「あさかつスタンプ」）。
2. メニュー **拡張機能 → Apps Script** を開く。
3. 最初からある `function myFunction() {}` を**すべて消して**、`gas/Code.gs` の中身を**まるごと貼り付け** → 💾 保存。
4. 上部の関数選択で **`setup`** を選び、**▶ 実行**。
   - 初回は権限の承認画面が出ます → 自分の Google アカウントを選び、「詳細」→「（プロジェクト名）に移動」→「許可」。
     （自分のスプレッドシートを自分で操作するだけなので安全です）
5. 実行が終わると、`members` / `stamps` / `config` の3シートが自動でできています。
   - **管理キーが自動生成**され、`config` シートの「管理キー」に入っています（メニュー **表示 → ログ** でも確認できます）。この文字列は後で使うのでコピーしておいてください。

---

## ステップ2: ウェブアプリとしてデプロイ

1. Apps Script 画面の右上 **デプロイ → 新しいデプロイ**。
2. 歯車⚙️ → **ウェブアプリ** を選択。
3. 設定:
   - **次のユーザーとして実行**: `自分`
   - **アクセスできるユーザー**: `全員`
4. **デプロイ** → 表示された **ウェブアプリのURL**（`https://script.google.com/macros/s/……/exec`）をコピー。

> ⚠️ あとでコードを直したときは、**デプロイ → デプロイを管理 → 編集（鉛筆）→ バージョン「新バージョン」→ デプロイ** で更新します（URL は変わりません）。

### 動作チェック（任意）

ブラウザのアドレスバーに次を貼って JSON が返ればOK（`あいことば` はサンプルの `りんご3`）:

```
＜ウェブアプリのURL＞?action=card&pass=りんご3
```

`{"ok":true,"name":"さんぷる1",...}` のように返れば成功です。

## ステップ3: 名簿と期間を本番用にする

スプレッドシートを開いて編集します（**再デプロイ不要**、次に開いたとき反映されます）。

### `members` シート（子どもの名簿）

サンプルの2行を、実際の子に書きかえ／追加します。1人1行。

| id | なまえ | あいことば |
|----|--------|-----------|
| 1  | はると | りんご3   |
| 2  | ゆい   | そらの5   |
| 3  | そうた | ねこ7     |

- **なまえ**: カードに表示される名前（ニックネームでOK）。
- **あいことば**: 子どもごとに配る合言葉。ひらがな＋数字など、他の子と重複しない短いものに。入れた子だけ自分のカードを開けます（他の子のカードは見えません）。
- **id**: 空でもOK（空なら「あいことば」を内部IDとして使います）。数字を入れておくと安心です。
- 置く個人情報は**ニックネームだけ**を推奨。

### `config` シート（設定）

`setup()` が入れた初期値を必要に応じて直します。

| key    | value                    |
|--------|--------------------------|
| 開始日  | 2026-07-21               |
| 終了日  | 2026-08-31               |
| 管理キー | （自動生成済み。基本そのままでOK） |
| タイトル | あさかつ スタンプカード     |

- **開始日・終了日**: カードのマス目の期間（夏休みなど）。`2026-07-21` の形式。`2026/7/21` でも読めます。
- **管理キー**: 管理ページを開く合言葉。自動生成済みなのでそのままでOK。自分で決めたい場合は推測されにくい長い文字列に。**子どもには渡しません**。
- **タイトル**: カード上部の見出し。

### `stamps` シート

**手で触らなくてOK**。管理ページのタップで自動的に行が増えます。

## ステップ4: URL をページに設定する

`index.html` と `admin.html` の両方の、先頭付近にあるこの1行を、ステップ2でコピーしたURLに書きかえます。

```js
var GAS_URL = 'ここにGASのURLをはりつける';
```

↓ 例

```js
var GAS_URL = 'https://script.google.com/macros/s/AKfycb……/exec';
```

---

## ステップ5: GitHub Pages で公開する

このリポジトリを GitHub Pages で公開すると、URL を配るだけで使えます。

1. GitHub のリポジトリ **Settings → Pages**。
2. **Source** を `Deploy from a branch` にし、Branch を公開したいブランチ（例: `main`）／フォルダ `/ (root)` にして **Save**。
3. 数分後、`https://＜ユーザー名＞.github.io/okane-to-suuji/` で公開されます。
   - 子ども向けカード: `.../okane-to-suuji/stamp/index.html`
   - 管理ページ: `.../okane-to-suuji/stamp/admin.html`

> 既存の算数ゲームはトップ（`/`）、スタンプカードは `/stamp/` で共存します。

---

## ステップ6: 配布する

- **子ども・保護者へ**: カードのURL（全員共通）＋ その子の**あいことば**を渡す。
  - 例:「`.../stamp/index.html` を開いて、あいことば『りんご3』を入れてね」
  - あいことばは端末に記憶されるので、2回目からは開くだけで自分のカードが出ます。
  - 保護者に個別リンクを渡したい場合は `?pass=りんご3` を付けたURLでもOK（入力不要で開きます）。
- **ぽこぴーさん（管理）**: 管理ページのURL `.../stamp/admin.html` ＋ **管理キー**。
  - スマホのホーム画面に追加しておくと朝活中にすぐ開けます。

---

## 毎朝の使い方

1. 朝活の Zoom を開始。
2. スマホで管理ページ（`admin.html`）を開く（初回だけ管理キーを入力）。
3. Zoom に来てくれた子の**名前をタップ** → その日のスタンプが付く（もう一度タップで取り消し）。
4. 押し忘れた日は、上の日付『‹ ›』で日を変えてから押せます。
5. 子どもがカードのURLを開くと、今日のスタンプが「ポン！」と押される演出＋効果音で表示されます。
   - 5個・10個…の節目で、種まきテーマのごほうびバッジ（🌱→🌿→🌷→🌸→🌳）が付きます。

## こまったとき

| 症状 | 確認すること |
|------|-------------|
| カードが「あいことばがちがう」になる | `members` シートの「あいことば」と入力が一致しているか（前後の空白に注意） |
| 管理ページが「管理キーがちがいます」 | `config` の「管理キー」と入力が一致しているか |
| 「つうしんに しっぱい」 | `GAS_URL` が正しく貼れているか。コード修正後に**新バージョンで再デプロイ**したか |
| スタンプが反映されない | `stamps` シートに行が増えているか。増えているのにカードに出ない場合は、その日付が `config` の開始日〜終了日の範囲内か |
| 期間を変えたい | `config` の開始日・終了日を編集（再デプロイ不要、次に開いたとき反映） |

---

## データについて

- 記録はすべて**あなたの Google スプレッドシート**の中にあります。いつでも手で確認・修正できます。
- スプレッドシートに置く個人情報は**ニックネーム（ひらがな）だけ**にする運用を推奨します。
- 管理キーは推測されにくい長い文字列にし、子どもには渡さないでください。
